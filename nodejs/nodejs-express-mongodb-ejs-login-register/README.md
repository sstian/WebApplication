
# 三级联动菜单

## Technology

- Node.js
- Express
- MongoDB - mongoose
- ejs
- cookie & session
- sha1

## Command

```cmd
npm init --yes

npm install express
npm install mongoose

npm install cors
npm install ejs

# read cookie
npm install cookie-parser

# operate session in express
npm install express-session
# MongoDB session store for Connect and Express written in Typescript
# mongoUrl
npm install connect-mongo

npm install sha1
```

## Font Awesome

https://www.runoob.com/font-awesome/fontawesome-tutorial.html

是一套的图标字体库和CSS框架。

提供可缩放矢量图标，可以被定制大小、颜色、阴影以及任何可以用CSS的样式。

<i>和 <span> 元素广泛用于图标。

更改图标容器的字体大小或颜色，图标会更改。

使用前缀 **fa** 和图标的名称来放置 Font Awesome 图标。
